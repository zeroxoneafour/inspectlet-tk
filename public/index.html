Inspectlet Developer Tools

Install the Inspectlet tools by adding the below link as a bookmark, then clicking it.

javascript:fetch('https://www.inspectlet.tk/inspectlet/inspectlet.txt').then((res) => res.text().then((t) => Function(t)() ))

Alternatively, use the prebuilt bookmarklet located here. This will work on many more sites.

For both of these bookmarklets, put the script in as the URL for a bookmark. Then click the bookmark to use Inspectlet.
